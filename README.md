# Fibonacci Series in C

## Objective
To implement the Fibonacci Series in C using both iterative and recursive approaches and demonstrate the difference between the two methods.

## Features
- Generate Fibonacci Series using Iteration
- Generate Fibonacci Series using Recursion
- Compare iterative and recursive approaches
- Simple console-based implementation

## Environment
- Programming Language: C
- Compiler: GCC (GNU Compiler Collection)
- Operating System: Windows / Linux / macOS
- Editor/IDE: VS Code, Code::Blocks, Dev-C++, Turbo C, or any text editor

## Requirements
- GCC Compiler or any C compiler
- Command-line terminal

## Sample Output

```text
Fibonacci Series (Iterative):

```

## Iterative Approach
- Uses loops to generate Fibonacci numbers.
- Faster execution.
- Requires less memory.

## Recursive Approach
- Uses function calls to generate Fibonacci numbers.
- Easier to understand mathematically.
- Consumes more memory and execution time.

## Conclusion
This project demonstrates the implementation of the Fibonacci Series using both iterative and recursive algorithms in C. The iterative approach is more efficient in terms of time and memory, while the recursive approach provides a simpler and more intuitive representation of the Fibonacci sequence.
